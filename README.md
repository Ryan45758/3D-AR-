# 將素材匯進Unity:<br>
接下來的步驟呢? 嗯...有點小複雜，總之就是無止盡的寫腳本，串接個張圖檔，點擊按鈕會觸發什麼東西之類的，在這邊我是將全景圖部屬到Unity Sphere的3D空間內，在3D空間內進行操作，這樣使用者看起來才會有3D的感覺，彷彿自身其中。實際怎實作，大家可以載來研究看看。這專案2018年做的，筆者現在研究領域跟Unity沒什麼關聯了，很久沒使用Unity拉 ~~ 可能不太熟悉了😂😂<br>
# 該如何快速把玩此專案呢?<br>
1. 將 "東海360(執行檔).exe" + "東海360(執行檔)_Data" + "UnityPlayer.dll" 這三個檔案下載下來放在同個目錄下<br>
2. 執行 "東海360(執行檔).exe" 會出現以下畫面 ，並點擊Play就可以玩拉 ~~~ <br>
<img src="readme圖庫/把玩.png" width="400"/><br>
3. 進入並載入完畢後底下有場景列表，總共分為17個場景，亦即有17張全景圖，如不需要場景列表可以點擊它將它關閉<br>
<img src="readme圖庫/場景列表.png" width="400"/><br>
4. 另外如果覺得背景音效太吵可以點擊右上角將音效關閉<br>
<img src="readme圖庫/音效.png" width="400"/><br>
# 檢討與改善:<br>
1. 圖與圖之間的拍照距離間隔過大，造成畫面移動有順移感。<br>
2. 切換場景時，動畫可以更加流暢。<br>
3. 效能可以提升，運行速度需更加流暢。<br>
4. 介面可以更加美化。<br>
